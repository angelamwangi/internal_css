Borders

Borders are like strokes in Figma.

Borders add border around your element.

It takes 3 properties to set a border.

  1. border-width
  2. border-style
  3. border-color

Border style:solid,dotted,dashed, double,groove,ridge,outset etc

Borders shorthand property doesn't matter which order you write with.width style color

Example:

border-width: 20px; border-style: solid; border-color: green; can be same as border:20px solid green.

You can also control borders of each side independetly.

hierarchy:margin-border-padding-content

margin,border and padding add to the width of an element to the left and right of it.
