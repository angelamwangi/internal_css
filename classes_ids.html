Classes and IDs are like elements, they are used to select somthing.

Classes (.)

HTML side:class="intro"

CSS side: .intro{style;}

IDs (#)

HTML side: id="intro"

CSS side: #intro{style;}

NB

IDs can only be used one time in a page while classes can be used over and over again.

When you select IDs and a Class at the same place,ID will override the class.
