Images are like links and they require an attribute for them to function.

Images use the src attribute which is short for source.

Basic Syntax

img src="Image.jpg" or, img scr="link.jpg"

however it is not considered good practice to link off images to external sites.

It is good to use alt attribute to describe the image eg: a very cute white cat in a bag.

If it is a decorative element or logo, it doesn't need alt text but it still requires the alt attribute eg: img src="logo.jpg" alt=""
